# Rake::Delphi

This gem adds an ability to compile automatically Delphi projects

## Installation

Add this line to your application's Gemfile:

    gem 'rake-delphi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rake-delphi
